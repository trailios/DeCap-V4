Introduction

This API provides endpoints for image classification and FunCaptcha solving. Use it to integrate advanced verification solutions into your workflow or application.

Base URL: http://178.254.44.95:10000

Endpoints

Classifies FunCaptcha images to determine the correct selection.

Methods: GET, POST

Parameter Type Description
variant String FunCaptcha variant.
imagebase64 String Base64-encoded image data.
prompt String (Optional) Additional text prompt for context.

Response: JSON object containing result (index and text).

Submits FunCaptcha challenges to be solved.

Method: POST

Parameter Type Description
key String API key.
method String Captcha site solving method.
proxy String Proxy (IP:port or user:pass@IP:port).
browser String Browser (e.g., chrome, firefox).
os String OS (windows, linux, mac).
version Integer Browser version.
blob String (Optional/required) Captcha data or metadata.

Response: JSON object indicating success/failure status.

Browser Support

Supported browsers for captcha solving:

  • Chrome
  • Edge
  • Firefox
  • Opera
  • Safari (macOS only)
